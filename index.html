使用文档

基本用法

在你的网站或GitHub README中添加以下代码:

<img src="https://your.domain.com/keyword1.svg?action=hit&title=访问量" />

参数说明

参数 可选值 说明
action view / hit view只显示计数，hit会增加计数
title 任意文本 徽标左侧显示的文字
count_bg %23后跟颜色代码 计数区域的背景颜色
title_bg %23后跟颜色代码 标题区域的背景颜色
edge_flat true / false true为直角，false为圆角
icon SimpleIcons图标名 在标题前显示的图标

JSON 格式

也可以获取 JSON 格式的计数数据:

https://your.domain.com/keyword1?action=hit

返回格式:

{ "counter": "keyword1", "action": "hit", "total": 1024, "daily": 64, "date": "2025-03-25", "timestamp": "2025-03-25T09:50:53.096Z" }
